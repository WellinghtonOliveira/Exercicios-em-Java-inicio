# Treinamento em Java

Este repositório é dedicado ao **treinamento da linguagem Java**, com foco na **resolução de problemas** e na **inicialização de projetos exclusivamente em Java**.

O objetivo principal é fortalecer a base da linguagem por meio de exercícios práticos, desafios de lógica e pequenos projetos, aplicando conceitos fundamentais como **lógica de programação**, **programação orientada a objetos** e boas práticas de desenvolvimento.

Este repositório é utilizado como ambiente de estudo contínuo, evolução técnica e consolidação dos conhecimentos adquiridos ao longo do aprendizado em Java.
